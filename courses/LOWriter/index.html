Программы для создания текстовых документов называются текстовыми процессорами.

LibreOffice Writer — часть свободно распространяемого пакета LibreOffice.

Любой текстовый процессор имеет базовые инструменты:

  • для работы со шрифтами: изменение типа, размера, начертания;
  • работы с файлом: сохранение, печать;
  • импорта объекта: вставка рисунков, таблиц, диаграмм;
  • форматирования текста: настройка параметров абзаца, отступов, межстрочного расстояния;
  • поиска и замены.

Инструменты сгруппированы в виде отдельных вкладок, расположенных в верхней части окна программы. Чтобы перейти к инструменту, пользователю необходимо выбрать нужную вкладку. Наиболее часто используемые инструменты чаще всего сгруппированы во вкладке Главная, которая открыта по умолчанию.

Панель инструментов

Назначение элементов панели инструментов понятно интуитивно.

Можно подвести курсор мышки к какому-либо инструменту и через некоторое время появится поянение об этом инструменте.

Ввод и редактирование текста

Чтобы удалить символ слева от курсора (мерцающая вертикальная черта), нажмите клавишу Backspace.
Чтобы удалить символ справа от курсора, нажмите клавишу Delete.
Для отмены предыдущего действия, удерживайте нажатой клавишй Ctrl и один раз нажмите z.

Курсор показывает, в каком месте документа будут появляться символы, вводимые с клавиатуры.

По умолчанию Writer работает в режиме вставки — при вводе весь текст справа от курсора сдвигается, чтобы освободить место для нового текста.

Форматирование текста

Форматирование текста используется когда требуется изменить начертание текста.
Сделать это можно с помощью кнопок, расположенных на панели Форматирование. Она включена по умолчанию, как и панель Стандартная.

Включение панелей инструментов доступно во вкладке Вид, пункт Панели инструментов.

  1. Выбор типа шрифта. В примере выбран тип Times New Roman.
  2. Выбор размера шрифта. В примере выбран размер 12 пунктов.
  3. Ж — полужирное начертание.
  4. К — курсив.
  5. Ч — подчёркивание.
  6. X2 — надстрочный символ.
  7. X2 — подстрочный символ.

Сохранение документа

  • Выберите в строке меню пункт Файл (File).
  • Выберите команду Сохранить (Save). На экране появится окно диалога Сохранение документа (Save As)
    Writer автоматически предлагает имя для документа (обычно Untitled 1). Введите любое другое имя

Альтернативный, более быстрый путь — комбинайия клавиш Ctrl + S

Форматирование абзацев

Для настройки абзаца необходимо вызвать соответствующее меню во вкладке Формат, установить настройки и подтвердить их нажатием кнопки OK.
Если абзац уже набран, то перед вызовом меню необходимо выделить его.
Таким образом, изменения будут применены сразу после нажатия кнопки ОК.

Стиль адзаца выбирается из стандартных стилей или создается в панели Стили и форматирование с помощью пункта контекстного меню «Создать...».

Пример: создание нумерованного заголовка на базе стиля Нумерованный заголовок 1

Во вкладке «Управление»:

  1. Задаем имя стиля «Нумерованный заголовок 1»;
  2. Устанавливаем автообновление;
  3. Устанавливаем следующий стиль «Основной текст»;
  4. Наследуем стиль от «Заголовок 1»;
  5. Помещаем стиль в категорию «Стили пользователя».

Во вкладке «Отступы и интервалы»:

  1. Устанавить отступ первой (красной) строки 1,25 см.
    Галку «Автоматически» не устанавливать, иначе Writer будет наследовать значения отступов из стиля «Заголовок 1».

Во вкладке «Шрифт»:

  1. Установим обычный «Liberation Serif» — это свободный аналог «Times New Roman».

Во вкладке «Положение на странице»:

  1. Включите автоматическую расстановку переносов.
    В LO Writer она устанавливается для каждого стиля индивидуально.

Ключевой является вкладка «Структура и нумерация».
На данной вкладке мы устанавливаем связь между создаваемым стилем «Нумерованный заголовок 1» и одним из предустановленных нумерованных списков.

Все нумерованные списки в Writer являются многоуровневыми, и нет необходимости создавать собственный многоуровневый список, достаточно для целей нумерации заголовков немного поправить структуру одного из стилей «Нумерованный список 1-5».

На вкладке «Структура и нумерация»:

  1. Устанавливаем стиль нумерации «Нумерованный список 1»;
  2. Нажмем на кнопку «Изменить стиль» и установим первые пять уровней, как мы делали это в Word: на каждом уровне в поле «Разделитель — После» убираем точку, и в поле «Все уровни» устанавливаем текущий уровень.

После этого по аналогии настраиваем стили «Нумерованный заголовок 2-5», наследуя их от стилей «Заголовок 2-5», и используя в качестве стиля нумерации уже настроенный стиль «Нумерованный список 1».

Аналогично настраиваем стиль «Структурный элемент отчета» с той лишь разницей, что в поле «Стиль нумерации» устанавливаем «Нет», а на вкладке «Эффекты шрифта» в поле «Эффекты» выбираем «Прописные буквы».

Вставка таблицы

Чтобы вставить таблицу в документ, необходимо перейти во вкладку Таблица и выбрать инструмент Вставить таблицу.

Выбрать необходимое количество строк и столбцов, стиль таблицы и подтвердить свой выбор нажатием кнопки Вставить.

Чтобы изменить размеры столбцов, необходимо поместить курсор внутрь таблицы и, перемещая маркеры таблицы на линейке сверху, добиться нужного размера.

Изменять размер строк и столбцов можно также перемещением границы, зафиксированной с помощью левой кнопки мыши.

Для выравнивания данных внутри строки нужно воспользоваться меню, расположенным внизу листа (строку предварительно надо выделить, удерживая правую кнопку мыши) и выбрать необходимый пункт для выравнивания ячеек.
