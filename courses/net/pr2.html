Настройка сервера имен

Технические рекомендации по выполнению

  1. Откройте терминал
  2. Получите адрес учебного компьютера
  3. Есть 2 пользователя: tux/tux и root/root
  4. Войдите на учебный компьютер "ssh tux@_адрес компьютера_"
  5. Для повышения полномочй команда "su -"
  6. Используйте редактор "nano" (или "vi")

Теоретическая часть

named.conf

options {
    directory "/etc/bind";
};
zone "." IN {
    type hint;
    file "named.ca";
};
zone "our.lan" IN {
    type master;
    file "named.our.data";
};
zone "33.22.10.in-addr.arpa" IN {
    type master;
    file "named.our.rev";
};

zone "0.0.127.in-addr.arpa" in {
    type master;
    file "named.our.local";
};

named.our.data

1                      IN    PTR    first.our.lan.
5                      IN    PTR    second.our.lan.
3                      IN    PTR    reception.our.lan.
7                      IN    PTR    comp18.our.lan.

named.our.rev

first        IN    A        10.22.33.1
second        IN    A        10.22.33.5
reception         IN    A        10.22.33.3
comp18      IN    A        10.22.33.7

named.our.local

zone "0.0.127.in-addr.arpa" in {
    type master;
    file "named.our.local";
};

named.our.local

$TTL 3h    ;3 hour
@ IN SOA our.lan. admin.our.lan.  (
                   1       ;serial
                   3600    ;refresh
                   600     ;retry
                   3600000 ;expire
                   3600    ;negative caching TTL
)
<tab>	IN    NS     our.lan.
1      IN    PTR    localhost.

resolv.conf

nameserver 127.0.0.1
domain our.lan

Задание

  • Настрить конфигурация named в соответствии с поставленной задачей
  • Настроить компьютер на использование локального DNS сервера
  • Определить при помощи утилиты nslookup адреса компьютеров uranus, mercury и www.crao.ru (скриншот приложить к отчету)

Поставка задачи

  1. Сервер конфигурируется в каталоге /etc/bind
  2. Сервер для домена solar.lan
  3. В домен входят хосты: mercury, venus, earth, mars, jupiter, saturn, uranus, neptune, pluto
  4. Адреса из пула 192.168.33.X/24
  5. DNS сервер расположен на mercury.solar.lan с адресом 192.168.33.1
